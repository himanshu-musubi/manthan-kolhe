Project information

  • Category: Compiler
  • Client: Techis, Bangalore
  • Project date: june, 2019

Developmet of compiler for U.S Students

Why?
Techis Platform is used by the student who joined techis school. The motto of this platform is to teach and help them to compile their code. It not only helps students,but makes them capable of learning programming languages as easily as possible. This project was inhouse project.

Technologies worked on?
Frontend:- HTML5, CSS3, javascript, jquery, ajax, bootstrap 4
Database:- MYSQL, Apache
Backend:- Python,Django
Deployment:- Github,Git

What it covers?
It covers multiple programming languages. A person can compile different languages like html, css, js and python. It also has an Admin dashboard for tracking student records.
